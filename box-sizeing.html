Understanding Box Sizing

The box-sizing property controls how the total width and height of an element is calculated.

content-box (default): Width and height only apply to the content area. Padding and border are added to the specified dimensions.

border-box: Width and height include content, padding, and border. The specified width/height is the total size of the element.

Key Observations:

  • With content-box, the actual size grows when you add padding or borders.
  • With border-box, the element maintains its specified dimensions, and the content area shrinks to accommodate padding and borders.
  • border-box makes layouts more predictable and is often set globally in CSS resets.

Global Reset Example:

html {
    box-sizing: border-box;
}
*, *:before, *:after {
    box-sizing: inherit;
}
